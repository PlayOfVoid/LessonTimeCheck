# LearnTimeCheck

## Описание проекта

**LearnTimeCheck** — это веб-приложение для управления расписанием занятий с автоматическими уведомлениями в Telegram.

### Основные возможности

- **Управление учениками**: Добавляйте учеников с именами (например: Тимофей(Юлия))
- **Расписание занятий**: Создавайте занятия для каждого ученика
- **Биография учеников**: Ведите подробную информацию о каждом ученике с поддержкой Markdown
- **Экспорт в PDF**: Скачивайте биографии учеников в формате PDF
- **Автоматические уведомления**: Получайте уведомления в Telegram за час и за 5 минут до начала занятия
- **Персонализация**: Выбирайте из 10 различных тем оформления
- **Безопасность**: Простая система авторизации с хешированием паролей

### Как это работает

1. Учитель входит в систему используя свой username и password
2. Добавляет учеников через форму
3. Для каждого ученика можно:
   - Создавать занятия с указанием времени
   - Заполнять биографию с Markdown разметкой
   - Скачивать биографию в PDF
4. Система автоматически отправляет уведомления в Telegram:
   - За 60 минут до начала занятия
   - За 5 минут до начала занятия
5. После уведомления за час занятие автоматически удаляется, но ученик остается в системе

### Технические детали

- **Backend**: Django 5.0.6
- **База данных**: SQLite
- **Telegram интеграция**: pyTelegramBotAPI
- **PDF генерация**: ReportLab
- **Markdown**: Python Markdown
- **Темы**: 10 различных вариантов оформления

### Настройка

- Telegram Chat ID можно указать в настройках аккаунта
- Время занятий указывается в московском времени (Europe/Moscow)
- Уведомления отправляются автоматически в фоновом режиме

### Разработка

Проект создан для удобного управления расписанием занятий и автоматизации напоминаний.
